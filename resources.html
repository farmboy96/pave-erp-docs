Azure Resources

The following table lists the Azure resources used by PAVE/ERPSense, their types, locations, resource groups, descriptions, and specific purposes in the product.

Name Type Location Resource Group Description Purpose in PAVE/ERPSense
ERPSense microsoft.web/staticsites eastus2 ERPSensePrototype Hosts static HTML/CSS/JavaScript files from a GitHub repository, like a website's default page. Serves PAVE/ERPSense documentation at https://purple-cliff-092ae700f.6.azurestaticapps.net, displaying System Overview, Key Features (Smartsheet, RAID logs, Blob Storage, Logic Apps), and Demo Instructions for consultants (April 17–19, 2025).
azureblob microsoft.web/connections eastus ERPSensePrototype Defines a connection to Azure Blob Storage for Logic Apps, storing credentials like a server script. Enables Logic Apps to access Blob Storage (p2pstorage123) to store RAID Log CSVs (raid_log.csv) and transcript attachments, acting as a bridge between Smartsheet and Azure storage (April 9–17, 2025).
p2pstorage123 microsoft.storage/storageaccounts eastus ERPSensePrototype Stores unstructured data (CSVs, PDFs, text) like a file server directory. Holds Smartsheet data (RAID Log CSVs, transcript PDFs, Infor guides) and AI outputs (embeddings, extracted text) for RAG, serving as the file cabinet for work products (April 10–17, 2025).
master microsoft.sql/servers/databases eastus ERPSensePrototype A SQL database storing structured data, like a website's backend database. A default or legacy database, possibly holding test data or metadata (e.g., Client_Master) from early setup, not actively used by PAVE/ERPSense (March 5, 2025).
PAVE-DB microsoft.sql/servers/databases eastus ERPSensePrototype A SQL database storing structured data, like a dynamic website's database. The primary database storing tables like Stakeholder_Master, RAID_Log, AI_Generated_WorkProduct, and AI_Metadata, supporting the Flask app’s PAVE Dashboard and RAG work products (e.g., BRE-140E) (April 9, 2025).
pave-sql-server microsoft.sql/servers eastus ERPSensePrototype Hosts SQL databases (master, PAVE-DB), like a database server. Manages connections for PAVE-DB, enabling the Flask app and Logic Apps to query RAID and stakeholder data for dashboards and reports (April 9, 2025).
smartsheet-search microsoft.search/searchservices eastus ERPSensePrototype An AI Search service indexing data for fast retrieval, like a website’s search engine. Indexes Smartsheet data (RAID Log, transcripts, Infor guides) in Blob Storage and Chroma, enabling RAG to retrieve data for work products like stakeholder reports and blueprints (April 10, 2025).
DefaultWorkspace-e800edc4-cfd8-4c8f-a6c0-1991a7eb3b6b-EUS microsoft.operationalinsights/workspaces eastus DefaultResourceGroup-EUS A Log Analytics Workspace collecting telemetry, like a server log file. Monitors Azure resources (Logic Apps, SQL Server) for performance and errors, aiding troubleshooting (e.g., Logic App failures) (April 11, 2025).
NetworkWatcher_eastus microsoft.network/networkwatchers eastus NetworkWatcherRG Monitors network performance, like a diagnostic tool for website traffic. Ensures network connectivity for data transfers (e.g., Smartsheet to Blob Storage), a default resource not directly used by PAVE/ERPSense (April 9, 2025).
pave-smartsheet-attachments microsoft.logic/workflows eastus ERPSensePrototype A Logic App automating workflows, like a server script processing uploads. Ingests Smartsheet attachments (transcript PDFs, PowerPoints), stores them in Blob Storage, and processes with Document Intelligence for RAG, supporting work products like action item summaries (April 16–17, 2025).
pave-smartsheet-sync microsoft.logic/workflows eastus ERPSensePrototype A Logic App automating data sync, like a script fetching API data. Syncs Smartsheet RAID Log rows (Risk, Action, Issue) to Blob Storage as raid_log.csv, providing data for RAG reports and dashboards (April 9, 2025).
MSVMI-DefaultWorkspace-e800edc4-cfd8-4c8f-a6c0-1991a7eb3b6b-EUS microsoft.insights/datacollectionrules eastus DefaultResourceGroup-EUS A Data Collection Rule for Azure Monitor, like a logging rule. Collects telemetry for resources, supporting diagnostics for Logic Apps and SQL Server, a default rule not directly used by PAVE/ERPSense (April 11, 2025).
p2p-language-service microsoft.cognitiveservices/accounts eastus ERPSensePrototype A Cognitive Services account for text analytics, like a text-processing API. Analyzes Smartsheet data (e.g., RAID comments, transcripts) for sentiment or key phrases, enhancing RAG outputs like risk prioritization (April 17, 2025).
smartsheet-doc-intelligence-s0 microsoft.cognitiveservices/accounts eastus ERPSensePrototype A Document Intelligence account extracting text from documents, like an OCR tool. Extracts text from Smartsheet attachments (transcript PDFs) using prebuilt-read, storing in Blob Storage for Chroma embeddings and RAG work products (April 10–11, 2025).
smartsheet-openai microsoft.cognitiveservices/accounts eastus ERPSensePrototype An OpenAI Service account for language models, like an AI content generator. Generates embeddings for Smartsheet data in Chroma and produces RAG outputs (e.g., blueprints, summaries), the brain for work products like BRE-140E (April 9–10, 2025).
smartsheet-doc-intelligence microsoft.cognitiveservices/accounts eastus ERPSensePrototype A Document Intelligence account, similar to smartsheet-doc-intelligence-s0. Processes additional Smartsheet attachments or serves as a test instance, supporting text extraction for RAG work products like meeting summaries (April 11, 2025).
rober-m9bbhzpg-swedencentral microsoft.cognitiveservices/accounts swedencentral ERPSensePrototype A Cognitive Services account, possibly for OpenAI or Document Intelligence. A test or secondary AI instance, possibly for EU compliance, processing Smartsheet data or Infor guides for RAG, not actively used (April 10, 2025).
